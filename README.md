# Vim Setups
---
## 1. Neobundle Installation
    $ cd ~
    $ curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh > install.sh
    $ $ sh ./install.sh
## 2. vim settings
    $ git clone https://github.com/riku-onuma/setups.git
    $ mv ./setups/.vimrc ./
    $ vim .vimrc
